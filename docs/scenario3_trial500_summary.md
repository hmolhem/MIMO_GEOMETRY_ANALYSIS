# Scenario 3 Trial-500 Summary

## 1. Purpose

This document summarizes the first controlled 500-trial Scenario 3 run after adding explicit ALSS command-line parameters to the paper experiment runner.

This run evaluates the selected ALSS calibration candidate:

```text
alss_mode  = ar1
alss_tau   = 0.25
alss_coreL = 3
```

The goal is to determine whether the selected ALSS setting provides stable RMSE improvement for the Z5 array under ideal and coupled conditions.
