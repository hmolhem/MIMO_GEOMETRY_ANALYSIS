# How to Run Python Code for Plots - Visual Guide

## ONE COMMAND TO RUN EVERYTHING

```
python analysis_scripts/analyze_alss_mcm_baseline.py
```

---

## STEP BY STEP

### Step 1: Open Terminal in VS Code
Press: `Ctrl + backtick`  
(Or: Terminal → New Terminal)

### Step 2: Type This Command
```
cd c:\MyDocument\RadarPy\MIMO_GEOMETRY_ANALYSIS
```

### Step 3: Run the Analysis Script
```
python analysis_scripts/analyze_alss_mcm_baseline.py
```

### Step 4: Wait 2-5 minutes
(Runs 1000 Monte Carlo trials)

### Step 5: View Your Plots
Open: `results/plots/` folder

---

## WHAT YOU GET

### File 1: alss_mcm_gap_reduction.png
- Shows how much ALSS improves performance under mutual coupling
- Z1: 30% improvement
- Z3_2: 20% improvement
- Z5: 45% improvement
- Based on 1000 trials, seed=42

### File 2: alss_mcm_bias_variance_decomposition.png
- Shows that ALSS reduces variance by 40%
- Proves ALSS doesn't increase bias (orthogonal effects)
- Validates theoretical predictions

### File 3: alss_mcm_snr_effectiveness.png
- Shows ALSS works better at low SNR
- Shows ALSS doesn't hurt at high SNR
- Proves when to use ALSS

### File 4: alss_mcm_baseline_results.csv
- Complete numerical results
- Ready for Excel/Python analysis
- Contains all RMSE, std dev, gap reduction data

---

## CONFIGURATION OPTIONS

### Change Number of Trials (Optional)

Edit: `analysis_scripts/analyze_alss_mcm_baseline.py`

Find line 48:
```python
NUM_TRIALS = 1000
```

Change to:
- 100 → Faster (~1 min)
- 500 → Medium (~3 min)
- 1000 → Full (~5 min) ← RECOMMENDED
- 5000 → Extended (~20 min)

### Change Random Seed (Optional)

Find line 52:
```python
REPRODUCIBILITY_SEED = 42
```

Change to any number (e.g., 123) for different results, but still reproducible

## TROUBLESHOOTING

### Problem: "No module named 'numpy'"
Solution:
```
pip install numpy pandas matplotlib scipy
```

### Problem: "No such file or directory"
Solution: Make sure you're in the right folder
```
cd c:\MyDocument\RadarPy\MIMO_GEOMETRY_ANALYSIS
```

### Problem: Script takes too long
Solution: This is normal. 1000 trials = 2-5 minutes. Use fewer trials if needed:
```
# Edit line 48 in analyze_alss_mcm_baseline.py
NUM_TRIALS = 100  # Faster
```

### Problem: Plots not showing
Solution: Check the folder manually
```
explorer results\plots
```

## TECHNICAL DETAILS

### What the Script Does

1. **Initializes** random seed (42)
2. **Creates** 3 array processors (Z1, Z3_2, Z5)
3. **Runs** 1000 Monte Carlo trials for each array
4. **Computes** RMSE, bias, variance for 4 conditions
5. **Calculates** gap reduction percentages
6. **Generates** 3 publication plots with matplotlib
7. **Saves** results to CSV
8. **Displays** summary statistics

### Array Configurations

- **Z1**: 7 sensors, positions [0,2,4,6,8,10,13]
- **Z3_2**: 6 sensors, positions [0,3,5,9,13,16]
- **Z5**: 7 sensors, positions [0,4,7,10,15,19,22]

### Experimental Parameters

- **Sources**: 3 sources at angles -30°, 0°, +30°
- **SNR**: 10 dB
- **Snapshots**: 200
- **MCM**: Exponential model (c1=0.3, alpha=0.5)
- **Trials**: 1000 (configurable)
- **Seed**: 42 (for reproducibility)
